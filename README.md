# SADANA (Sistem Aplikasi Dana Penjualan)

A professional Point of Sale (POS) and Financial Management System built with **Flutter** and **MySQL**.

## Features
- **Dashboard**: Sales trends and quick statistics.
- **Cashier**: Product selection, cart management, and PDF receipt printing.
- **Product Management**: Custom inventory for any business (Cafe, Warung, Retail).
- **History**: Detailed transaction records and revenue summary.

## Project Structure
- `lib/`: Flutter source code (Android Studio ready).
- `_server/`: Node.js backend and MySQL schema.

## Getting Started
1. Run MySQL and import `_server/database.sql`.
2. Start the backend: `cd _server && npm install && node index.js`.
3. Open the root folder in **Android Studio**.
4. Run the app on an Android Emulator or Device.
